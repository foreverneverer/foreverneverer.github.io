# Paperily GPT 隐私政策

欢迎使用 Paperily GPT。我们非常重视您的隐私，并致力于保护您的个人信息。本隐私政策旨在帮助您了解我们收集哪些数据、如何使用这些数据以及您拥有的权利。本政策适用于 Paperily GPT 浏览器扩展程序（包括 Edge 和 Chrome 版本）及相关服务。

## 1. 数据收集与使用

Paperily GPT 仅在为您提供服务所必需的范围内收集和使用数据。我们收集的数据类型及用途如下：

### 1.1 网页内容（PDF 数据）
*   **收集内容**：当您在浏览器中打开 PDF 文件并主动使用 Paperily GPT 的“解析”功能时，我们会读取当前页面的 PDF 文本内容。
*   **用途**：用于发送给我们的 AI 模型进行深度分析、摘要生成和翻译。
*   **存储**：解析后的摘要和分析结果会存储在我们的服务器上，以便您后续查看和进行研讨。原始 PDF 文件本身不会被永久存储，除非您主动将其保存到“资源共享”库中。

### 1.2 用户账户信息
*   **收集内容**：如果您选择注册登录，我们会收集您的用户名、电子邮件地址和加密后的密码。
*   **用途**：用于识别您的身份、同步您的阅读历史、收藏夹和评论数据。

### 1.3 交互数据
*   **收集内容**：您在插件内的点击操作、搜索记录、评论内容和收藏行为。
*   **用途**：用于改善用户体验、优化产品功能以及为您推荐相关的学术资源。

## 2. 数据保护与安全

我们采取行业标准的安全措施来保护您的数据：
*   所有数据传输均通过 HTTPS 加密协议进行。
*   您的密码经过高强度哈希加密存储，我们无法查看您的明文密码。
*   我们严格限制对服务器数据的访问权限，仅限授权人员进行维护。

## 3. 数据共享

**我们不会将您的个人数据出售给任何第三方。**仅在以下情况下，我们可能会共享您的数据：
*   **法律要求**：如果法律法规要求或为了配合司法程序，我们可能需要披露您的信息。
*   **服务提供商**：我们可能会使用受信任的第三方服务提供商（如云托管服务）来协助我们运营服务，这些提供商仅有权在代表我们履行职责时处理您的数据，并受保密协议约束。

## 4. 用户权利

根据相关法律法规，您拥有以下权利：
*   **访问权**：您有权访问我们存储的关于您的个人信息。
*   **更正权**：如果您的信息不准确，您有权要求更正。
*   **删除权**：您可以随时联系我们要注销账户并删除您的所有个人数据。

## 5. 针对 Microsoft Edge 用户的特定说明

本扩展程序遵守 Microsoft Edge 扩展商店的隐私政策要求：
*   **透明度**：我们在收集任何数据前均会征得您的同意（例如，插件仅在您点击解析按钮后才开始处理数据）。
*   **最小化原则**：我们仅请求实现功能所必需的权限（如读取当前标签页的 PDF 内容），不会滥用权限收集无关信息。

## 6. 变更通知

我们可能会不时更新本隐私政策。任何重大变更都会在本页面上发布，并在扩展程序中通知您。继续使用我们的服务即表示您同意受修订后的隐私政策约束。
